# Text Components
`React Slide Craft`では、スライドに表示するテキストには純正の`Text Components`を使用してください。
そして、すべての`Text Components`は、この`CustomText`を継承しています。

## Text Componentsの特徴
`|`で文字列を囲うと、囲われた文字列が強調されます。また、強調される文字列に独自のスタイルをあてることができます。

強調のスタイルで`highlightFontSize`という強調時のフォントサイズを指定するプロパティがありますが、`*`の後に数字を指定するとデフォルトのフォントサイズに指定した数値をかけたものを設定できます。(ex. `*1.1` → `size * 1.1`)

また、`\n`で改行可能です。

<Canvas of={Texts.Highlight} />

### 指定可能なsize
`size`はTextのfont sizeを指定するプロパティで、対応している単位は下記の通りです。（`px`指定は`px`なしでも指定可能です）
- `px`
- `rem`
- `em`
- `%`
- `auto`

また上記の単位で指定できる値以外に、指定できる独自の値があります。
