# TitleWithBgAndBodySlideTemplate
`TitleWithBgAndBodySlideTemplate`は、背景色付きのタイトルセクションと本文を持つスライドテンプレートです。タイトルを視覚的に強調したい場合に最適です。

## Usage
```tsx
import { TitleWithBgAndBodySlideTemplate } from "react-slide-craft"

const AccentSlide = () => (
  <TitleWithBgAndBodySlideTemplate
    title="重要なセクション"
    body="このスライドでは、タイトルが背景色で強調されます。"
    titleBgColor="#3b82f6"
    titleTextColor="white"
  />
)
```

## 特徴
- **強調されたタイトル**: 背景色でタイトルを際立たせるデザイン
- **カスタマイズ可能**: タイトルの背景色と文字色を自由に設定
- **視覚的インパクト**: セクションの区切りや重要なポイントを明確に表現
- **統一感**: 本文部分は通常のレイアウトを維持
