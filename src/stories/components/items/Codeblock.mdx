# Codeblock
`Codeblock`は、スライド内でソースコードを美しく表示するためのコンポーネントです。シンタックスハイライト機能と豊富なカスタマイズオプションを提供します。

## Usage
```tsx
import { Codeblock } from "react-slide-craft"

const MySlide = () => (
  <div>
    <Codeblock
      language="javascript"
      theme="dracula"
    >
      {`function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}`}
    </Codeblock>
  </div>
)
```

## 特徴
- **185の言語サポート**: JavaScript、Python、TypeScript、Java、C++など幅広い言語をサポート
- **87のテーマ**: ダークテーマからライトテーマまで豊富なカラーテーマ
- **ウィンドウコントロール**: macOSライクなウィンドウコントロールを表示可能
- **カスタムスタイリング**: フォントサイズ、角丸、シャドウなどを自由に調整
- **レスポンシブ**: スライドのサイズに応じて適切にスケーリング

## サポート言語（一部）
- **Web技術**: JavaScript, TypeScript, HTML, CSS, JSON, YAML
- **プログラミング言語**: Python, Java, C++, C#, Go, Rust, Swift
- **スクリプト言語**: Bash, PowerShell, Ruby, PHP
- **データベース**: SQL, PostgreSQL
- **マークアップ**: Markdown, XML
- **設定ファイル**: Docker, Nginx, Apache

## 人気のテーマ
- **ダークテーマ**: `dracula`, `atomOneDark`, `gruvboxDark`, `nord`, `monokai`
- **ライトテーマ**: `github`, `atomOneLight`, `vs`, `tomorrow`
- **特色あるテーマ**: `synthwave84`, `shadesOfPurple`, `nightOwl`

## 使用上の注意
- `children`にはコードを文字列として渡してください
- テンプレートリテラル（`` `code` ``）を使用することで複数行コードを簡潔に記述できます
- `language`プロパティを正しく設定することで、適切なシンタックスハイライトが適用されます
- `useFontSize`フックにより、フォントサイズは自動的にスライドサイズに調整されます

## パフォーマンス
- react-syntax-highlighterを使用しているため、大きなコードファイルの場合はパフォーマンスに注意してください
- 長いコードの場合は、必要な部分のみを抜粋することを推奨します
