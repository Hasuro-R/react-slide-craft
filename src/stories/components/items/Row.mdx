# Row
`Row`は、子要素を横並びに配置するレイアウトコンポーネントです。水平方向のレイアウト構築に使用します。

## Usage
```tsx
import { Row } from "react-slide-craft"

const MySlide = () => (
  <Row>
    <div>左側のコンテンツ</div>
    <div>中央のコンテンツ</div>
    <div>右側のコンテンツ</div>
  </Row>
)
```

## 特徴
- **水平レイアウト**: 子要素を横並びに配置
- **フレキシブルな配置**: justify-content、align-itemsなどの調整が可能
- **レスポンシブ対応**: 画面サイズに応じた適切な配置
- **簡単な使用方法**: childrenを渡すだけで横並びレイアウトを実現
